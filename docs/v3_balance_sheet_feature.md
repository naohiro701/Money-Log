# v3 新規提案: 家庭バランスシート機能

## 提案コンセプト
家計簿（フロー）だけではなく、資産・負債（ストック）を定点観測し、
**「純資産の改善サイクル」**を回せるようにする。

## 良い方法（実装した方式）
1. レシート由来の支出データは従来どおり蓄積。
2. 月次または任意タイミングで、ユーザーが資産/負債を入力してスナップショット保存。
3. 純資産を自動計算（総資産 - 総負債）。
4. 支出データの月平均と直近純資産を使って改善提案を返す。

## v3 実装ポイント
- UI: `/balance-sheet`
  - 資産入力（現金、投資、不動産、その他）
  - 負債入力（住宅ローン、カード/ローン、その他）
  - 次月アクションプラン記録
- API:
  - `GET /api/balance-sheet/snapshots`
  - `POST /api/balance-sheet/snapshots`
  - `GET /api/balance-sheet/recommendation`
- 保存先: `Flask/data/balance_sheet_snapshots.json`

## 期待効果
- 純資産が見えることで、支出削減だけでなく資産形成の意思決定がしやすくなる。
- 「何を買ったか」から「家計体質がどう変わったか」まで追跡できる。
